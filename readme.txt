Adds an option in the Easy Digital Downloads discount edit screen to apply a discount to the signup fee of a recurring payment.

== Description ==

**Signup Fee Discount for EDD** enhances Easy Digital Downloads by adding the ability to apply discounts to the signup fees of recurring payments. This is particularly useful for memberships or subscriptions with one-time setup fees.

### Features:
- Toggle to apply discounts to signup fees.
- Supports percentage and flat-rate discounts.
- Fully compatible with Easy Digital Downloads recurring payments.

### Use Case Example:
If your recurring product has a signup fee of $100 and the customer applies a 10% discount, this plugin ensures the discount applies to both the recurring amount and the signup fee.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/signup-fee-discount-edd` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Edit or create a discount in Easy Digital Downloads.
4. Use the "Apply discount to signup fee" toggle to enable the feature.

== Frequently Asked Questions ==

= Does this plugin work with all versions of Easy Digital Downloads? =  
This plugin is designed for Easy Digital Downloads 3.0+ and requires the Recurring Payments add-on.

= Does the plugin affect non-recurring products? =  
No, the plugin only applies the discount to signup fees for recurring payments.

= How do I enable a discount on the signup fee? =  
Edit or create a discount in Easy Digital Downloads, and toggle the "Apply discount to signup fee" option.
